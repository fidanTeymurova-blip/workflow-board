# WorkFlow Board

Kanban-style workflow management application built with Vanilla JavaScript.

## Features

* Drag & Drop task management
* Event-driven architecture
* LocalStorage persistence
* Responsive design
* CRUD operations
* Zero dependencies

## Tech Stack

* HTML5
* CSS3
* Vanilla JavaScript (ES6+)
* Browser APIs
* LocalStorage API
* Drag & Drop API
